# Fast MCP Servers

- [Fast MCP V3](https://gofastmcp.com/getting-started/welcome)

## アプリケーションの仕様

LlamaIndex の日本語翻訳を自動化する際に、あえて MCP を導入してみる。  
MCP 導入におけるメリットの1つが、エージェント別に使いまわせる「再利用性」だが、今回はあくまで MCP の導入が目的になるので、そこは無視しておく。

### ザックリ設計

1. 翻訳元のリソース(ローカルリポジトリ)から、最後に翻訳したバージョンと最新版のバージョンで比較して、更新対象となるファイルの一覧を取得する。  
    - MCPとして情報を提供する。
    - 参照ファイルに対する更新先も提示しておく。
2. 一覧から選択したファイルを日本語に翻訳する。
    - 日本語翻訳では LLM モデルを切り替える。
3. 翻訳結果の内容に納得できたら、ドキュメント管理側のリソースを更新する。
    - ファイルの更新も MCP として機能を提供する。
    - 翻訳結果が気に入らない場合は、指示コメントと共に再編集させたい。

第1弾はここまで対応したい。

## Environment

```bash
# 初期化
uv init
# venv の作成
uv venv --python 3.14
# パッケージのインストール
uv sync
```

## Run

```bash
uv run fastmcp run app/main.py:mcp --transport http --port 8000
```
